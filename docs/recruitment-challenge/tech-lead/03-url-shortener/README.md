# 🔗 Desafio 03: URL Shortener & Analytics (Tech Lead)

Este desafio avalia a capacidade de liderar o desenvolvimento de um **Encurtador de URLs e Plataforma de Rastreamento de Métricas** com foco em otimização de leitura usando caches distribuídos, estruturação limpa do código do time e gerenciamento de débito técnico.

---

## 🗺️ O Pipeline de Contratação

1. **[Etapa 1: Recruiter Phone Screen](./01-recruiter-screening.md)**
   * *Foco:* Comunicação didática, coordenação técnica de equipe e resolução de conflitos.
2. **[Etapa 2: Technical Screening](./02-technical-screening.md)**
   * *Foco:* Redirecionamentos HTTP (301 vs 302), codificação Base62 básica e noções de cache de rede.
3. **[Etapa 3: System Design Onsite](./03-system-design-onsite.md)**
   * *Foco:* Arquitetura do **Redirection & Analytics Engine**, cache distribuído (Redis) e processamento analítico assíncrono básico.
4. **[Etapa 4: Coding Onsite](./04-coding-onsite.md)**
   * *Foco:* Implementação de encurtador concorrente em memória com controle de colisões de chave e legibilidade.
5. **[Etapa 5: Leadership Onsite](./05-leadership-onsite.md)**
   * *Foco:* Priorização de refatoração técnica, governança de código de equipe e negociação de escopo de produto.
