# 🚗 Desafio 02: Ride-Sharing Dispatcher (Tech Lead)

Este desafio avalia a capacidade de liderar o desenvolvimento de um **Serviço de Despacho e Alocação de Motoristas** com foco em concorrência thread-safe estruturada, monitoramento operacional de equipe (observabilidade) e design de APIs de integração robustas.

---

## 🗺️ O Pipeline de Contratação

1. **[Etapa 1: Recruiter Phone Screen](./01-recruiter-screening.md)**
   * *Foco:* Mediação de conflitos intelectuais no time, mentoria ativa e adaptabilidade a cenários dinâmicos.
2. **[Etapa 2: Technical Screening](./02-technical-screening.md)**
   * *Foco:* Indexação espacial básica, APIs assíncronas baseadas em WebSockets/Polling e noções de concorrência.
3. **[Etapa 3: System Design Onsite](./03-system-design-onsite.md)**
   * *Foco:* Arquitetura do **Dispatcher Engine** do time, cache geoespacial com Redis e telemetria operacional da equipe.
4. **[Etapa 4: Coding Onsite](./04-coding-onsite.md)**
   * *Foco:* Programação concorrente thread-safe estruturada e legível para a equipe em fila de tarefas espaciais.
5. **[Etapa 5: Leadership Onsite](./05-leadership-onsite.md)**
   * *Foco:* Planejamento técnico da sprint do time, mitigação de bugs em produção e negociação de débito técnico.
