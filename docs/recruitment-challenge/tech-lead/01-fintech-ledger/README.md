# 💳 Desafio 01: FinTech Event-Driven Ledger (Tech Lead)

Este desafio avalia a capacidade de liderar o desenvolvimento de um **Sistema de Conciliação e Ledger Financeiro** de um time de pagamentos, com foco em arquitetura orientada a eventos, padrões de design limpos e divisão de trabalho.

---

## 🗺️ O Pipeline de Contratação

1. **[Etapa 1: Recruiter Phone Screen](./01-recruiter-screening.md)**
   * *Foco:* Gestão de pessoas técnica, mentoria, processos ágeis (Scrum/Kanban) e fit cultural.
2. **[Etapa 2: Technical Screening](./02-technical-screening.md)**
   * *Foco:* Padrões SOLID, isolamento de transações de banco de dados, design de APIs REST/gRPC e mensageria básica.
3. **[Etapa 3: System Design Onsite](./03-system-design-onsite.md)**
   * *Foco:* Arquitetura do **Ledger Event-Driven** integrado com filas (Kafka/RabbitMQ), consistência local e observabilidade.
4. **[Etapa 4: Coding Onsite](./04-coding-onsite.md)**
   * *Foco:* Refatoração de código seguindo Clean Architecture, SOLID e concorrência básica segura.
5. **[Etapa 5: Leadership Onsite](./05-leadership-onsite.md)**
   * *Foco:* Divisão técnica de projetos (Slicing), estimativas, gestão de débitos técnicos com produto e resolução de conflitos de time.
