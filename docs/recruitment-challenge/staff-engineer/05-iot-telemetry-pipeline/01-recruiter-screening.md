# 📞 Trilha 5 - Etapa 1: Recruiter Phone Screen (Data Platform)

* **Responsável:** Gaby (Tech Recruiter)
* **Duração Recomendada:** 30 a 45 minutos
* **Foco:** Liderança técnica de times de engenharia de dados, capacidade de priorização sob restrições orçamentárias e foco no valor do dado para a empresa.

---

## 🎯 Perguntas Comportamentais (Metodologia STAR)

### 1. Comunicação de Valor de Infraestrutura de Dados (15 min)
* **Pergunta:** "Infraestrutura de dados e analytics é frequentemente vista por diretores financeiros apenas como um centro de custo invisível. Pode compartilhar uma situação em que você teve que negociar orçamento extra para infraestrutura ou governança de dados (ex.: migração de Data Lake ou novo cluster Flink)? Como você justificou o valor econômico desse projeto para diretores não-técnicos?"
* **Sinal Verde (Green Flag):** Vincula a qualidade e latência do dado à eficiência de tomada de decisão, redução de fraudes ou novos produtos de dados para clientes finais.
* **Sinal Vermelho (Red Flag):** Justifica apenas com argumentos de "pureza técnica" (ex.: "mudar porque a tecnologia X é mais nova").

### 2. Mentoria e Disseminação da Cultura de Dados (15 min)
* **Pergunta:** "Como Staff Engineer de dados, seu papel também é incentivar outros times de engenharia de software a gravarem dados de forma limpa e padronizada nas origens. Como você influenciou outros times a adotarem práticas rígidas de contratos de dados (Data Contracts)?"

---

## ⚖️ Rubrica de Avaliação de Gaby
* **Aprovado (Staff):** Perfil altamente focado em alinhamento de negócios; compreende que o dado deve servir a um propósito estratégico produtivo.
