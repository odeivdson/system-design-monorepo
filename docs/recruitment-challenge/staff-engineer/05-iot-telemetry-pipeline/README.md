# 📟 Trilha 5: IoT Telemetry & Analytics Pipeline (Big Data & Data Platform)

Esta trilha avalia habilidades em processamento de dados massivos em fluxo (*Stream Processing*), gestão de concorrência e resiliência de canais de dados (Backpressure) em pipelines analíticos de larga escala.

---

## 🗺️ O Pipeline de Contratação

1. **[Etapa 1: Recruiter Phone Screen](./01-recruiter-screening.md)**
   * *Responsável:* Gaby.
   * *Foco:* Liderança de times de plataforma de dados e alinhamento com stakeholders de produto/negócios analíticos.
2. **[Etapa 2: Technical Screening](./02-technical-screening.md)**
   * *Responsável:* Senior Engineer.
   * *Foco:* Fundamentos de processamento distribuído (MapReduce, Spark, Flink), estratégias de particionamento de brokers (Kafka) e armazenamento colunar (Parquet/Iceberg).
3. **[Etapa 3: System Design Onsite](./03-system-design-onsite.md)**
   * *Responsável:* Staff Engineer (Alex).
   * *Foco:* Projeto de arquitetura da **Plataforma de Ingestão e Processamento em Tempo Real de Telemetria de Dispositivos IoT**.
4. **[Etapa 4: Coding & Stream Onsite](./04-coding-stream-onsite.md)**
   * *Responsável:* Staff Engineer.
   * *Foco:* Implementação de uma janela de tempo deslizante (*sliding window*) concorrente para agregação de métricas com tratamento de dados atrasados (*late events*).
5. **[Etapa 5: Leadership & Systemic Impact Onsite](./05-leadership-systemic-impact.md)**
   * *Responsável:* Director of Engineering.
   * *Foco:* Gestão de débito técnico em pipelines Spark/Flink herdados de equipes dissolvidas e governança de dados em Data Lakehouses.
