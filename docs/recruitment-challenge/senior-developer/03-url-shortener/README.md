# 🔗 Desafio 03: URL Shortener Caching (Dev Senior)

Este desafio avalia a capacidade de implementar um **Encurtador de URLs** funcional com estratégias de cache distribuído, algoritmos de compressão determinísticos e tratamento semântico de erros HTTP.

---

## 🗺️ O Pipeline de Contratação

1. **[Etapa 1: Recruiter Phone Screen](./01-recruiter-screening.md)**
   * *Foco:* Trajetória individual, autonomia para propor soluções arquiteturais e colaboração em equipe.
2. **[Etapa 2: Technical Screening](./02-technical-screening.md)**
   * *Foco:* Redirecionamentos HTTP 301 vs 302, representações numéricas (Base62) e tratamento de erros de APIs REST.
3. **[Etapa 3: System Design Onsite](./03-system-design-onsite.md)**
   * *Foco:* Arquitetura do **Redirecionador Web** local, cache de leitura/gravação NoSQL e consistência de cache.
4. **[Etapa 4: Coding Onsite](./04-coding-onsite.md)**
   * *Foco:* Algoritmo de compactação Base62 com cache e manipulação eficiente de strings/buffers.
5. **[Etapa 5: Behavioral Onsite](./05-leadership-onsite.md)**
   * *Foco:* Feedback técnico construtivo, ownership de features e comunicação em time.
