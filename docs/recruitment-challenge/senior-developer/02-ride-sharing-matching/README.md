# 🚗 Desafio 02: Ride-Sharing Matcher Básico (Dev Senior)

Este desafio avalia a capacidade de implementar um **Serviço de Localização e Pareamento de Veículos** simples em tempo real, com foco em concorrência thread-safe local, uso de mapas em memória e design de endpoints HTTP limpos.

---

## 🗺️ O Pipeline de Contratação

1. **[Etapa 1: Recruiter Phone Screen](./01-recruiter-screening.md)**
   * *Foco:* Trabalho colaborativo, resolução de problemas e adaptabilidade cultural em Big Techs.
2. **[Etapa 2: Technical Screening](./02-technical-screening.md)**
   * *Foco:* Noções de Geohash, transporte de rede (REST APIs) e concorrência na linguagem escolhida.
3. **[Etapa 3: System Design Onsite](./03-system-design-onsite.md)**
   * *Foco:* Arquitetura de microsserviço de **Pareamento Local de Motoristas** e armazenamento geoespacial rápido.
4. **[Etapa 4: Coding Onsite](./04-coding-onsite.md)**
   * *Foco:* Algoritmo de busca concorrente em memória de motoristas por vizinhança geoespacial.
5. **[Etapa 5: Behavioral Onsite](./05-leadership-onsite.md)**
   * *Foco:* Pareamento técnico educativo, superação de prazos e mentoria de novos integrantes do time.
