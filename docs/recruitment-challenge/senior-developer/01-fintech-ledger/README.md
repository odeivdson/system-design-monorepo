# 💳 Desafio 01: FinTech Ledger Local (Dev Senior)

Este desafio avalia a capacidade de implementar um **Ledger Financeiro e Sistema de Saldo Local** consistente, com foco em atomicidade de banco de dados relacional, prevenção de concorrência e testes unitários ricos.

---

## 🗺️ O Pipeline de Contratação

1. **[Etapa 1: Recruiter Phone Screen](./01-recruiter-screening.md)**
   * *Foco:* Trajetória de contribuição técnica individual, resolução de bugs complexos e fit cultural.
2. **[Etapa 2: Technical Screening](./02-technical-screening.md)**
   * *Foco:* Primitivas de concorrência na linguagem escolhida, modelagem de banco de dados e APIs REST funcionais.
3. **[Etapa 3: System Design Onsite](./03-system-design-onsite.md)**
   * *Foco:* Arquitetura de microsserviço de **Ledger e Saldo Consistente** local usando transações ACID.
4. **[Etapa 4: Coding Onsite](./04-coding-onsite.md)**
   * *Foco:* Implementação limpa de processamento de pagamentos concorrente com locks locais e testes mocks.
5. **[Etapa 5: Behavioral & Teamwork Onsite](./05-leadership-onsite.md)**
   * *Foco:* Mentoria a desenvolvedores juniores, resolução de disputas técnicas pessoais e trabalho colaborativo.
